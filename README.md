# Password Generator

## Description

Today, we spend more and more time on the internet for our works, entertainments and our personal stuff. To secure all of the information, we might have lots of different accounts for every single website services. Sometimes, it might cause you a headache when a website asking you to create a security enough password to sign up a member account with them.

And this password generator tool helps you to avoid that headache by choosing how long of the password you want, and what type of characters you want to include to meet the website's requirement.

Instead of setting a password related to your personal information, this random password generator is able to create an irregular password and is strong enough to protect your account safety.

Friendly reminder: Please write it down to make sure you won't forget your password!
